Números Reais: Conjuntos Numéricos, Operações e Propriedades

Conjuntos Numéricos

    Naturais (N): Números usados para contagem, como 0, 1, 2, 3, ... Inteiros (Z): Incluem todos os números naturais e seus opostos negativos, como ..., -2, -1, 0, 1, ... Racionais (Q): Números que podem ser escritos como uma fração, como 1/2, 3/4, ou números inteiros expressos como fração (por exemplo, 5 = 5/1). Irracionais: Números que não podem ser escritos como fração, como √2 e π. Eles possuem infinitas casas decimais não periódicas. Reais (R): Incluem todos os conjuntos acima: naturais, inteiros, racionais e irracionais.

Operações

Nos conjuntos numéricos, as operações fundamentais são:

    Adição Subtração Multiplicação Divisão (com exceção de divisão por zero, que é indefinida)

Propriedades

    Comutativa: A ordem dos números não altera o resultado.
      Exemplo: a + b = b + a e a × b = b × a
    Associativa: A maneira de agrupar os números não altera o resultado.
      Exemplo: (a + b) + c = a + (b + c) e (a × b) × c = a × (b × c)
    Elemento Neutro:
      Para adição, o elemento neutro é 0, pois qualquer número somado a 0 permanece o mesmo: a + 0 = a. Para multiplicação, o elemento neutro é 1, pois qualquer número multiplicado por 1 permanece o mesmo: a × 1 = a.

Números Reais: Conjuntos Numéricos, Operações e Propriedades

Conjuntos Numéricos:

  • Naturais (N): Números usados para contagem, como 0,1,2,3,…
  • Inteiros (Z): Incluem todos os números naturais e seus opostos negativos, como …,−2,−1,0,1,...
  • Racionais (Q): Números que podem ser escritos como uma fração
  • Reais (R): Incluem todos os conjuntos acima.

Operações: Adição, subtração, multiplicação e divisão (exceto por zero).

Propriedades:

  • Comutativa: a ordem não altera o resultado
  • Associativa: a forma de agrupar não altera o resultado
  • Elemento Neutro: Para a adição é o 0 (qualquer número somado a 0 permanece o mesmo) e para a multiplicação é o 1.

Equações do 1º Grau: Resolução de Equações e Inequações Simples

Uma equação do 1º grau é uma equação que pode ser expressa na forma ax + b = 0, onde a e b são números reais e a ≠ 0. O objetivo é encontrar o valor de x.

Uma inequação é uma expressão que compara dois valores usando sinais de desigualdade (>, <, ≥, ≤).

Funções: Conceito de Função

Uma função é uma relação entre dois conjuntos, onde cada elemento do primeiro conjunto (domínio) está associado a exatamente um elemento do segundo conjunto (contradomínio).

É frequentemente representada como f(x), onde x é a variável independente e f(x) é a variável dependente. Funções do 1º grau têm a forma geral f(x) = ax + b, onde a e b são constantes, e a ≠ 0.

Geometria Plana: Teorema de Pitágoras, Propriedades de Triângulos, Polígonos e Circunferências

Teorema de Pitágoras: Aplica-se a triângulos retângulos, afirmando que o quadrado da hipotenusa é igual à soma dos quadrados dos catetos.

Propriedades de Triângulos:

  • Equilátero: Todos os lados e ângulos iguais.
  • Isósceles: Dois lados iguais.
  • Escaleno: Todos os lados diferentes.

Sistemas de Medidas: Conversões entre Unidades de Medida, Cálculo de Áreas e Volumes

Comprimento: Conversão entre unidades como metros, centímetros e milímetros. Exemplo: 1 m = 100 cm = 1000 mm.

Massa: Conversão entre quilogramas, gramas e miligramas. Exemplo: 1 kg = 1000 g.

Volume: Conversão entre litros, mililitros e metros cúbicos. Exemplo: 1 L = 1000 mL.

Estatística: Medidas de Tendência Central (Média, Mediana e Moda)

Média: Soma dos valores dividida pelo número de elementos.

Mediana: Valor que divide um conjunto de dados ordenados em duas partes iguais.

Moda: Valor que aparece com mais frequência em um conjunto de dados.
